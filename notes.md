生成文件命令

```sh
# 生成整个模块
nest g resource xxx

# 使用 nest-cli 提供的指令可以快速创建文件，语法如下
nest g [文件类型] [文件名] [文件目录（src目录下）]
nest g module user logical
nest g service user logical
nest g controller user logical
```
